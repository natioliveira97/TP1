Descrição detalhada

Armazena um cep e sua cidade correspondente.

Construtores e Destrutores

◆ CEP() [1/2]

CEP::CEP ( )

Construtor default.

Inicializa os atributos cep com valor zero e cidade como string vazia.

◆ CEP() [2/2]

CEP::CEP ( int  )

Construtor que inicializa a classe com valor para cep.

Para ser válido, o cep passado deve pertencer às faixas de valores correspondentes às seguintes cidades:

  • Sao Paulo: 1000000-5999999 e 8000000-8499999
  • Rio de Janeiro: 20000000-26600999
  • Brasilia: 70000000-70999999
  • Salvador: 40000000-41999999
  • Fortaleza: 60000000-60999999

Exceções
