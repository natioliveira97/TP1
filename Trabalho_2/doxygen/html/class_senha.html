Senha::Senha ( std::string  )

Construtor que inicializa a classe com valor para senha.

A senha deve estar no formato XXXXXX onde cada X é dígito (0 – 9) e não há dígito repetido.

Parâmetros
senhaString a ser escrita no atributo senha.
Exceções
invalid_argumentse senha não estiver no formato especificado.
